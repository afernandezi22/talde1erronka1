◆ login()

login (   $json)

LOGIN-a egiteko funtzioa. POST bidez jasotako JSON-ean erabiltzaile izena eta pasahitza lortzen dira. Horiek datu-basean konparatzen dira eta horren arabera beste JSON bat bueltatzen da front-era informazio zehatzarekin. Erabiltzailea eta pasahitza ondo idatzi diren eta ondo egin bada beste informazio gehiago gehituko zaio: erabiltzailearen irudiaren esteka (null bada lehenetsitako izango da), erabiltzaile izena, rola eta izena. Informazio guzti hori gero front-ean erabiliko da cookia sortzeko.
